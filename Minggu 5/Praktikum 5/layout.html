Tutorial semantik Elemen untuk pemula

Semantik element itu adalah element yang memiliki makna dan tujuan. Tujuannya adalah agar kode HTML mudah dibaca dan tidak ada penyalahgunaan tag. Element semantik bagus untuk SEO dan juga dapat meningkatkan accessibility.
